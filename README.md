# 성형외과학 문제은행

GitHub Pages로 배포하는 정적 HTML 문제은행입니다.

## 구성

- `index.html`: GitHub Pages 루트 진입점
- `성형외과_문제은행/index.html`: 문제은행 메인 페이지
- `성형외과_문제은행/bank_A.html` - `bank_D.html`: 각 문제은행 페이지
- `성형외과_문제은행/css/style.css`: 공통 스타일
- `성형외과_문제은행/js/app.js`: 선택지 클릭, 채점, 필터, 이미지 모달 로직
- `성형외과_문제은행/js/data.js`: 문항 데이터
- `성형외과_문제은행/assets/question_figures/`: 문항 시각자료
- `성형외과_문제은행/assets/explanations/`: 해설 이미지
